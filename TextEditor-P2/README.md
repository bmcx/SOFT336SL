# Text Editor Part 2
Created with `c++` and `Qt`
#### Part 2 Features:
 - Add an application icon
 - Editor font customizations
 - Undo/Redo actions
 - Cut, Copy and Paste actions
 - Action keyboard shortcuts
 - About dialogs for application and Qt framework
